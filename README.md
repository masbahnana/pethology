# 🐾 Pethology

[![Cypress E2E Tests](https://github.com/masbahnana/pethology/actions/workflows/cypress.yml/badge.svg)](https://github.com/masbahnana/pethology/actions/workflows/cypress.yml)

Pethology is a free study platform built for animal care students in Ireland. It helps students prepare for their PLC exams through quizzes, smart review, and progress tracking — accessible on any device, no app required.

## Stack

- **Frontend** — HTML, CSS, vanilla JavaScript
- **Auth** — Auth0
- **Database** — Firebase Firestore (REST API)
- **Hosting** — Netlify

## Features

- 10 quiz modules covering Biology, Animal Welfare, Grooming, Anatomy, and more
- Exam Mode — timed, fullscreen, no hints
- Smart Review — automatically targets your weakest module
- Adaptive Quiz — adjusts difficulty based on your performance
- Student dashboard with stats, achievements, and progress tracking
- Teacher dashboard with announcements and class management
- 3D Anatomy viewer (coming soon)
